btErrorImg
directive in module boomstrap

Description

The btErrorImg directive will replace the src of an img tag if the original src fails to load. This is useful for images of properties.

Usage

as attribute
<img bt-error-img="{string}">
   ...
</img>

Parameters

ParamTypeDetails
btErrorImgstring

URL to a backup image to be used if the src of an img tag fails to load.
