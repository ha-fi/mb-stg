btDropdown
directive in module mineralbay

Description

btDropdown allows you to create a dropdown based on an object. The keys of the object are what is stored in the ngModel and the values are the visual representations in the dropdown. In most cases, prefer the ui-select directive over this.

Dependencies

  • ngModel

Usage

as attribute
<ANY bt-dropdown
     values="{Object}"
     keys-are-numbers="{Boolean}"
     on-assign="{expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
valuesObject

Object containing the values for the dropdown.

keysAreNumbersBoolean

Indicates that the keys of the values object are Numbers. This is used for sorting because by default iterating over an object will sort based on the string representation. Thus, 10 will come before 2, etc...

onAssignexpression

Expression to call when a value is chosen from the dropdown.
