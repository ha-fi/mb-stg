btScore
directive in module boomstrap

Description

The btScore directive represents a score component from Boomstrap. It will keep the color and size up to date based on what is specified.

Usage

as element:
<bt-score
       score="{Number}"
       size="{string}">
</bt-score>

Parameters

ParamTypeDetails
scoreNumber

The score to present in the component. Classes will be added to the element to color a score element based on how high or low the score is. Excellent score (99-76); Good score (75-56); Average score (50-26); Default score (25-0)

sizestring

The size of the score. Valid sizes include: xs (extrasmall), sm (small), and lg (large). If no size is specified, the score will be of medium size.

(default: '')
