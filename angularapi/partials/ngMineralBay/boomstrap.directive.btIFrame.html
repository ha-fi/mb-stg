btIFrame
directive in module boomstrap

Description

The btIFrame attribute directive allows the user to embed an iFrame with the ability to communicate the iFrame closing back to Angular. The iFrame must be on the same domain as the calling code, and must implement a global function called registerClose, which takes a function as an argument and sets a function internally to that function. It is intended that that function will be called when the iFrame wishes to close.

Dependencies

  • $window

Usage

as attribute
<ANY bt-i-frame
     width="{string}"
     height="{string}"
     src="{string}"
     close-frame="{expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
widthstring

Number value representing the width of the iFrame

heightstring

Number value representing the height of the iFrame

srcstring

The uri to the iFrame

closeFrameexpression

Expression to call when the iFrame has been closed internally.
