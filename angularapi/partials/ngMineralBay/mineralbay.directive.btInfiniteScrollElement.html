btInfiniteScrollElement
directive in module mineralbay

Description

The btInfiniteScrollElement attribute directive allows the user to trigger an expression when the user has scrolled to the bottom or close to the bottom of an element. This causes the illusion of infinite scroll.

Dependencies

  • $rootScope
  • $timeout

Usage

as attribute
<ANY bt-infinite-scroll-element="{expression}"
     infinite-scroll-distance="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
btInfiniteScrollElementexpression

Expression to evaluate when the bottom of the element has been scrolled to.

infiniteScrollDistancestring

Number value representing how far from the bottom of the element should the btInfiniteScrollElement expression be triggered.
