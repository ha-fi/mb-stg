btFocusOn
directive in module boomstrap

Description

The btFocusOn attribute directive will focus on an input when the expression provided goes from false to true. This is helpful when guiding the user through a page.

Dependencies

  • $timeout

Usage

as attribute
<ANY bt-focus-on="{expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
btFocusOnexpression

Expression that will be evaluated to truthy or falsey. If this expression goes from falsey to truthy, focus will be placed on the element
