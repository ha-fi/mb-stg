btPager
directive in module boomstrap

Description

The btPager directive is a wrapper of the ui-bootstrap pagination directive that keeps track of what page we are on and programmatically disable or enable pagination based on such. It is wrapping the directive so that we can provide our own template.

Usage

as element:
<bt-pager
       total-items="{Number}"
       current-page="{Number}"
       items-per-page="{Number}">
</bt-pager>

Parameters

ParamTypeDetails
totalItemsNumber

The number of items to paginate through.

currentPageNumber

The number of the current page we are on.

itemsPerPageNumber

How many items are allowed on each page.
