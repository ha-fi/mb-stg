btAutoSubmit
directive in module mineralbay

Description

btAutoSubmit allows you to auto submit a form whenever the model of a form-field changes. This directive requires the ngForm directive to also be present on the element.

Dependencies

  • $timeout
  • ngForm

Usage

as attribute
<ANY bt-auto-submit
     ng-submit="{expression}"="{expression}"
     submit-timeout="{Number}"
     will-load="{expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
ngSubmitexpression

Fires upon timeout completion

btAutoSubmitexpression

If the expression is truthy then the ngSubmit expression will fire upon timeout

submitTimeoutNumber

Number of milliseconds to wait after a model change before firing the ngSubmit

willLoadexpression

This directive will set the value supplied here to true when it has scheduled the timeout, but not fired it
