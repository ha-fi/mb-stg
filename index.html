About this Pattern Library

The present style guide and pattern library is built on the Bootstrap framework - with MineralBay branding and custom, reusable components. Especially with such a large and broad target group, it is important to create a recognition value through uniform design and recurring design. In addition to the actual design, particular importance is also given to UX guidelines.
